# Dagger
This Model is created for game design,which is a low poly.

# Software Specification
1. Autodesk Maya (For creating the base mesh).
2. Z brush (For Creating the High Poly).
3. Marmoset ToolBag
4. Substance Painter. (For Texturing the mesh)
